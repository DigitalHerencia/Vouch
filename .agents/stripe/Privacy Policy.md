Vouch (“we,” “us,” “our”) respects your privacy, mostly because regulators insist on it.

## 2. Information We Collect

### 2.1 Information You Provide

When you use Vouch, you may provide:

- Name
- Email address
- Phone number
- Government-issued ID (for verification)
- Payment account details (via third-party providers)

If you type it in, we can see it. Shocking, I know.

---

### 2.2 Identity Verification Data

To keep things slightly less chaotic, we may collect:

- ID documents
- Selfie / biometric verification data
- Age confirmation

This is processed through third-party providers (e.g., Stripe Identity or similar), not stored for fun.

---

### 2.3 Transaction Data

We collect:

- Payment amounts
- Transaction status (pending, released, refunded)
- Timestamps

We do **not** collect or store:

- The purpose of your meeting
- Details of what you’re doing

We don’t want to know. Truly.

---

### 2.4 Location Data

We may collect location data to:

- Confirm proximity between users
- Enable presence-based release logic

This may include:

- GPS coordinates
- Proximity events

We use it for functionality, not to stalk you.

---

### 2.5 Device & Usage Data

Like every other app on earth:

- IP address
- Device type
- Browser info
- Log data

Mostly so things don’t break.
